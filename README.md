# Billiards Strength & Conditioning

Static single-page PWA for tracking orthodox billiards training with:

- Daily minimums: Stop Shot Ladder and Long Straight-In
- Weekly rotation: L-Drill, Rail Cut Drill, Mother Drill, Sunday Combine
- Form-over-result logging with separate mechanical and potting failure paths
- Mechanical audit prompt after 3 consecutive failures
- Focus Score and weekly summary review table

Data is stored locally in the browser under the `bsc-restoration-v1` localStorage key.
